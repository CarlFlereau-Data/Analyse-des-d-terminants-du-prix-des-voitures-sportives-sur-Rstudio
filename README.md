# Analyse des déterminants du prix des voitures sportives sur Rstudio
Projet d’analyse statistique en R visant à identifier les facteurs influençant le prix des voitures sportives. À partir d’un dataset Kaggle, l’étude explore les relations entre performances (puissance, moteur, accélération) et prix via analyse descriptive, tests statistiques et régressions linéaires.
